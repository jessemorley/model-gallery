# Model Gallery

A simple proof of concept web app to browse model portfolios.

## Features
- Lazy loading of model cards
- Search and filter by agency
- Click to cycle portfolio images

## Folder structure
- `models/`: JSON data for each agency
- `index.html`: Main HTML
- `script.js`: App logic
- `style.css`: Styling
